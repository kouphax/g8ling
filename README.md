# Installing/Running

    > git clone https://github.com/kouphax/giter8-catalog.git
    > cd giter8-catalog
    > bundle install
    > bundle exec middleman

    Browse to http://localhost:4567

    Deploy/Build with

    > bundle exec middleman build

    Generated site goes to ./build
